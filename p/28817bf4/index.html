移除日志中的图片 Base64

发表于 2022-05-09 更新于 2022-05-30 分类于 Java Web 阅读次数： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

有的时候在日志里看到图片 Base64，一整串又臭又长的字符串，严重影响了日志的可读性。这里提供了一个方法，可以将图片 Base64 移除。

Image Base64 Pattern：[A-Za-z\\d+/]{214}([A-Za-z\\d+/][A-Za-z\\d+/=]+|==)
